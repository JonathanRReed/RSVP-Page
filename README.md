# Try Noko IRL — RSVP Page

A minimal landing page for a sparkling drink event. Contains a semantic HTML form styled with a modern dark theme.

## Files

- `index.html` — Markup for the landing page
- `styles.css` — Page styles (no JS required)
- `.gitignore` — Ignores common local/dev files

## Quick start

1. Open `index.html` in your browser.
2. Optionally serve locally for auto-reload:
   - Python: `python3 -m http.server`
   - Node (if installed): `npx serve .`

## Customize

- Replace the logo text inside `.brand__logo` in `index.html`.
- Adjust theme colors in `styles.css` under the `:root` variables.

## Accessibility

- Labels are associated with inputs via `for`/`id`.
- Button has an explicit `aria-label`.
